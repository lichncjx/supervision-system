# 测试发布检查清单

## 发布前检查

- [ ] CI 是否通过
- [ ] Docker 镜像是否构建成功
- [ ] 是否涉及数据库迁移

## 部署检查

- [ ] 是否执行 migrate.sh
- [ ] 是否执行 deploy.sh
- [ ] docker-compose ps 状态是否正常
- [ ] supervision-app 是否 running
- [ ] supervision-db 是否 running

## 功能验证

- [ ] 页面是否能打开
- [ ] /api/health 是否正常
- [ ] 核心账号是否能登录
- [ ] 本次修改涉及页面是否验证
- [ ] 不同角色权限是否验证

## 验收闭环

- [ ] 测试通过后是否关闭 Issue
